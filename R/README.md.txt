The files in the folder contain all the code, that was used in the different experiments.

In order to run the code one must install the following software:

R (Version 4.1.0)
Keras (Version 2.4.0)
Tensorflow (Version 2.5.0)

As the datasets were too large too upload, they are not included but available under:

https://github.com/kevinsaner91/Master-Thesis

The easiest way to run the code is to check out the whole repository. In order to run, some file paths might be updated.
